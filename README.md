# Setup of Polythene 1.0 for Mithril

Compatible with Mithril 1.x

Minimal setup example how to create a [Polythene](https://github.com/ArthurClemens/polythene) app for Mithril.


## Setup

init:

~~~
yarn
~~~

start development server (port 3000):

~~~
yarn run dev
~~~

build:

~~~
yarn run build
~~~
